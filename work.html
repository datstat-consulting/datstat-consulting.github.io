Selected Projects

Marawi City Econometric Study

Client was a government organization that requested demand forecasting. Services included barangay population forecasting, land use forecasting, traffic modeling, and social service demand. Study used inverse distance weighting to interpolate missing data, and Random Forest regression for projections.

Cohesion and Prestige Social Network Analysis

Client requested DATSTAT for consultation over calculating Degree Cohesion and Degree Prestige, then designing study for analysis. Chief Statistician gave advice on survey design, proposed a Multilevel Model analysis framework, and proposed a novel way to calculate group Degree Prestige. Study completed to client's satisfaction.

IMU Accelerometer Signal Processing

Client requested DATSTAT to perform signal processing on data obtained from wristworn IMU units to detect hand tremors. Chief Statistician applied Butterworth Filter and Fast Fourier Transform to obtain tremor data, then built a script for automated tremor detection.

Illiquid Assets Pricing Model

Client requested DATSTAT to build a model for pricing illiquid assets. Said asset was cryptocurrency with token lockups, requiring illiquid asset analysis. Chief Statistician implemented a Geometric Brownian Motion model developed by Ang, Papanikolaou, and Westerfield (2014) using Python. Implementation consisted of applying It√¥'s lemma to obtain analytical formula for illiquid asset, estimating illiquid asset Pearson correlation with liquid asset, and Monte Carlo price simulation. Chief Statistician also encountered fat-tailedness, requiring use of Laplacian Distribution in Monte Carlo. Model obtained 97% accuracy in backtesting.

State Space Model using Kalman Filter for Macroeconomic Forecasting

Client requested DATSTAT to perform Kalman filtering in Python for Macroeconomic and Financial forecasting. However, existing Python libraries supported only non-panel data. Chief Statistician thus performed a novel implementation of Kalman filtering that supported panel data with exogenous control variables. Task was accomplished in a week.

Traffic forecasting in Software Defined Networking using Deep Learning

Client requested DATSTAT to implement four Deep Learning algorithms in Python for use with internet traffic data to predict which website a user would go to based on packets and other parameters. Chief Statistician implemented the algorithms using the Scikit-learn library for Python. Project was accomplished in a week.

Classification Model for Shark Behavior

Client was an Australian Biologist who wanted DATSTAT to predict shark behavior from accelerometer data. Additionally, client requested for panel logistic modeling to determine probabilities of behavior. Chief Statistician performed analyses in R and Stata to great feedback. Project was accomplished in a month.

Uncorrelated Capital Asset Pricing Estimation for Multiasset Portfolio

Client requested DATSTAT to estimate a Capital Asset Pricing Model for multiasset portfolio. Both alphas and betas needed to be uncorrelated. Chief Statistician performed the analysis with the plm library in R to determine each asset's alpha and beta, and to test whether multiasset model had greater likelihood than pooled model. Project was accomplished in a month.
